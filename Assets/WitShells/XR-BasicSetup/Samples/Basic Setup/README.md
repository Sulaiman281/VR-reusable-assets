This sample demonstrates:
- Driving an Animator's `Trigger` and `Grip` floats from `InputActionProperty`.
- Toggling controller visuals when controllers are tracked, and hiding them when hands are tracked.

How to use
- Import this sample from the Package Manager under "WitShells XR Basic Setup".
- Open the provided scene or drop the provided prefab into your own scene.

Notes
- Bind `triggerAction` and `gripAction` to your controller scheme in the Input System.
- Ensure XR Hands is enabled and an XRHandSubsystem is active.
